# ☀️ SUJANA SUN OS v12.0
Official Low-Level Operating System and Compiler built by **Sujana Tech**.

## 🚀 Features
- **Pure SUN Syntax:** Powered by `ray`, `add`, and `shine`.
- **Hardware Simulation:** Virtual CPU Accumulator and 16-slot RAM.
- **Persistence:** Save/Restore system states to `memory.bin`.
- **Ubuntu-style Shell:** A clean, "Aura Green" terminal interface.

## 🛠️ How to Run
1. Download `SujanaOS.exe`.
2. Double-click to boot the Kernel.
3. Type `ray x = 100` then `store 1` to interact with hardware.

## 📜 Language Specs
- **ray**: Set CPU Register
- **add**: ALU Addition
- **store/load**: Memory Management
- **save/restore**: Disk Persistence
